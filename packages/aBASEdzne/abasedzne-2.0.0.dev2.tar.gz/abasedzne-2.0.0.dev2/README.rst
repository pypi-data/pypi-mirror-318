=========
aBASEdzne
=========

Overview
--------

This project is the adaptation of aBASE from the CodeOcean capsule. See the original paper for more information.

Installation
------------

To install ``aBASEdzne``, you can use ``pip``. Open your terminal and run:

.. code-block:: bash

    pip install aBASEdzne
