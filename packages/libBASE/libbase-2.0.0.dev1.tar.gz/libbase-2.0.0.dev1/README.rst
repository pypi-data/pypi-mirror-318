=======
libBASE
=======

Overview
--------

This project is the adaptation of libBASE from the CodeOcean capsule. See the original paper for more information.

Installation
------------

To install ``libBASE``, you can use ``pip``. Open your terminal and run:

.. code-block:: bash

    pip install libBASE
